## Welcome to Auth.js

Auth.js is a comprehensive open-source authentication solution for web applications. Explore our live demos to see Auth.js in action:

### Key Features

- Built-in support for <b>{Object.keys(manifest.providers).length}+</b> popular services (Google, Facebook, Auth0, Appleâ€¦)
- Built-in support for <b>{manifest.adapters.length}+</b> databases/ORMs (MySQL, Postgres, Prisma, Drizzleâ€¦)
- Built-in email/passwordless/magic link authentication
- Compatible with any [OAuth 2](https://www.ietf.org/rfc/rfc6749.html) or [OpenID Connect](https://openid.net/specs/openid-connect-core-1_0.html) provider
- Compatible with any username/password store

### Flexible Integration

- Runtime agnostic - deploy anywhere! Vercel Edge Functions, Node.js, Serverless, etc.
- Compatible with any modern framework! Next.js, SolidStart, SvelteKit, etc.
- [Bring Your Own Database](/getting-started/adapters) - MySQL, Postgres, MSSQL, MongoDB, etc. Choose database sessions or JWT.

### Security First

- Signed, prefixed, server-only cookies
- Built-in CSRF protection
- No reliance on client-side JavaScript
- Encrypted JWT sessions
